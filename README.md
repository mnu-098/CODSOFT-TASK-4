# 📒 Contact Book Application

A simple and interactive **Contact Book Application** built using Python. This project helps users efficiently manage their contacts through a command-line interface.

---

## 🚀 Features

✨ Create New Contacts

👀 View Contact Details

✏️ Update Existing Contacts

🗑️ Delete Contacts

🔍 Search Contacts by Name

📊 Count Total Contacts

❌ Exit Application

---

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Loops
- Conditional Statements
- Functions of CRUD Operations
- User Input Handling

## ⚙️ How It Works

The application stores contact details in a Python dictionary.

Each contact contains:
- Name
- Age
- Email
- Mobile Number

The program provides a menu-driven interface where users can perform different operations.
